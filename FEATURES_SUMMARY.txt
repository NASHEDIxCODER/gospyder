✨ NEW CLI FLAGS

-format string
    Output format: txt, json, csv (default "txt")
    Usage: gospyder -d example.com -ports -format json -o results.json

-json
    Quick JSON output flag (shorthand for -format json)
    Usage: gospyder -d example.com -ports -json -o results.json

-retry int
    Number of retries for failed connections (default 2)
    Usage: gospyder -d example.com -ports -retry 3

-v
    Verbose output (shows retry attempts, timing info)
    Usage: gospyder -d example.com -ports -v
